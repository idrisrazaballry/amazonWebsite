🛒 Amazon Homepage Clone

A front-end clone of the Amazon homepage built using HTML5 and CSS3.
This project replicates the basic layout and UI design of Amazon including the navigation bar, search functionality, product sections, and footer.

The project focuses on improving frontend development skills and UI layout design using Flexbox.

🚀 Features
✔ Amazon-style Navigation Bar
✔ Search bar with category dropdown
✔ Location and delivery section
✔ Shopping cart icon
✔ Hero banner section
✔ Product category boxes
✔ Responsive Flexbox layout
✔ Footer with multiple information sections

🧰 Technologies Used
HTML5
CSS3
Font Awesome Icons
Flexbox Layout

1️⃣ Navigation Bar
Includes:
Amazon Logo
Delivery location
Search bar
Sign in section
Orders section
Cart icon

2️⃣ Search Section
Includes:
Dropdown for product categories
Search input field
Search icon
Hover effect adds an orange border similar to Amazon UI.

3️⃣ Hero Section
Displays a large banner image with a message directing users to Amazon India.
